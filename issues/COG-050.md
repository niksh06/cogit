# COG-050: Graph queries over premises — taint, then support

Priority: P2 (after COG-049; entry criterion: real journals actually
contain premises — measure adoption first, COG-044-style)

## Scope

- `taint <assertion-id | source-uri>`: reachability closure over the
  reversed premise graph — every conclusion resting on the tainted
  assertion (or on any assertion from the tainted source). Output:
  affected assertions with their claims, ready to feed a recall
  cascade (remove-fact with reason). The primitive the trust line
  (COG-026) needs.
- `support <assertion-id>`: maximin evidence strength — max over
  derivation paths of the min confidence along the path (bottleneck
  path). Reported per conclusion with the limiting premise named.
- Placement: start as integrations analytics (same staging as
  COG-045/047); core-CLI promotion once query shapes stabilize.

## Acceptance criteria

- Synthetic fixture graphs (diamond, chain, multi-source) yield exact
  closures and maximin values; live-journal run documented; adoption
  metric (share of assertions carrying premises) reported alongside.

## Resolution (2026-07-11)

Shipped at the spec'd placement (integrations, like analytics/lint):
`derivation.py` with `taint` (reversed-premise closure; seed = assertion
id/prefix OR source-uri substring; depth-ordered rows ready for a
refute/retire cascade; `--history` reaches inactive dependents, each row
flagged `active`) and `support` (maximin/widest-path: max over
derivation paths of the min confidence; returns the winning chain and
the bottleneck assertion). Cycles impossible by construction — no cycle
guard needed. Both exposed as MCP tools (29 total); adoption metric
reported inside every taint result.

Live run 2026-07-11: adoption 28/446 active (6.3%); a real aleph chain
gave support 10000 bps over 2 links, and tainting its evidence leaf
covered 14 downstream conclusions — the graph already answers recall
questions at today's adoption. Entry criterion note: owner pulled this
forward; D-rules + write-time hints are expected to raise adoption.
