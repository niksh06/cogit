# COG-065: ceremony cut — MCP defaults for source and confidence

Priority: P2 (field feedback: full schema per fact is heavier than a
markdown append)

## Resolution

MCP `add_fact`/`record` facts require only kind/subject/predicate/object
(+ project on shared journals by convention). Defaults: `source` =
`agent:<instance-actor>` (attributable, Rule 10-consistent), and
`confidence_bps` = 9000 ("stated" band) with `tool_observation` = 9900
(observation band). Lint R4 still nudges deliberate outliers; explicit
values always win. CLI is unchanged (scripting keeps explicitness).

## Acceptance criteria

- A 4-field fact lands via MCP with source agent:<instance> and the
  band default for its kind; explicit source/confidence override.
